Tutorial

Number of players: Use the arrows to adjust the number of players to any integer between 1 and 4 inclusive.


Calculator

Word: Enter the word you would like to score (case does not matter)> Use '_' to denote a blank letter.

Scheme: 'n' = normal, no muliplication on this square; 'd' = double letter; 'D' = double word; 't' = triple letter; 'T' = triple word.


Controls

Add points: Will add the points calculated to the player whose turn it is.

Next player: The turn ends and it is now the next player's turn. The current player is colored blue.

Update scoreboard: The scores on the scoreboard will be updated with the newest points added.

GAME FLOW: Input number of players, start game → Submit words and scheme → Add Points → Update Scoreboard → Next Player
